# Desafío 2: Chatbot Basado en LSTM para Preguntas y Respuestas

    ## Descripción
    Este desafío consiste en construir un chatbot que responda preguntas utilizando un modelo basado en LSTM con embeddings de FastText. 
    El objetivo es permitir al chatbot responder preguntas simples de manera coherente, utilizando técnicas de NLP y deep learning.

    ## Contenido
    - **desafio_2.ipynb**: Jupyter Notebook con el código y el entrenamiento del chatbot.

    ## Requisitos
    - Python 3.x
    - Jupyter Notebook
    - Librerías: torch, numpy, pandas, gensim, matplotlib

    ## Instalación de dependencias
    Ejecuta el siguiente comando para instalar todas las dependencias necesarias:
    ```bash
    pip install -r requirements.txt
    ```

    ## Cómo ejecutar
    1. Abre el archivo `desafio_2.ipynb` en Jupyter Notebook.
    2. Ejecuta las celdas en orden para entrenar el modelo y probar el chatbot.

    ## Resultados obtenidos
    - El chatbot logra generar respuestas básicas con un desempeño aceptable tras el entrenamiento.
